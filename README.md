# Speed Typing Game

## Run Locally 

Clone the project  

~~~bash  
git clone https://github.com/sulekaptan/speed-typing-game.git
~~~

Go to the project directory  

~~~bash  
cd speed-typing-game
~~~

Install dependencies  

~~~bash  
npm install
~~~

Start the project  

~~~bash  
npm start
~~~
